# Project Summary

# Enterprise Linux Infrastructure Lab

## Overview

This project demonstrates the deployment of a small enterprise Linux
environment using Ubuntu Server 24.04 LTS and Oracle VirtualBox.

The environment was designed to showcase practical Linux administration,
infrastructure, networking, automation and troubleshooting skills
suitable for entry-level Infrastructure Engineer, Systems Administrator
and Linux Administrator roles.

## Environment

-   Hypervisor: Oracle VirtualBox
-   Server: LNX-SRV01
-   Client: LNX-CL01
-   Private Network: EnterpriseLab (192.168.100.0/24)

## Technologies Implemented

-   Ubuntu Server 24.04 LTS
-   SSH
-   Samba
-   NFS
-   Apache2
-   UFW Firewall
-   Netplan
-   Bash
-   Cron
-   Systemd
-   Journalctl

## Key Deliverables

-   Multi-VM Linux infrastructure
-   Static IP configuration
-   Secure remote administration
-   Cross-platform file sharing
-   Linux-to-Linux NFS sharing
-   Web server deployment
-   Automated health reporting
-   Automated configuration backups
-   Network diagnostics
-   System monitoring
-   Log analysis

## Skills Demonstrated

-   Linux administration
-   Infrastructure deployment
-   Service configuration
-   Network troubleshooting
-   Automation with Bash
-   Scheduled tasks with Cron
-   Security with UFW
-   Documentation
-   Validation and testing

## Outcome

The project successfully delivered a fully functioning enterprise-style
Linux lab with documented installation, configuration, validation,
troubleshooting and automation. It forms part of a professional GitHub
portfolio intended to demonstrate practical infrastructure engineering
skills to employers.
